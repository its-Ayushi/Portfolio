Projects

Traviya – Unified Travel Planning Platform

Traviya is a dynamic, web-based travel planning application built to simplify the travel experience by offering a unified digital platform. Despite the proliferation of travel tools and services, users often find themselves switching between multiple apps or websites for different travel needs—such as booking flights and hotels, creating itineraries, joining travel groups, or contacting local travel agents. This fragmented experience not only consumes time but also leads to inconsistent planning and confusion. Traviya bridges this gap by consolidating all these services into a single, browser-accessible application that ensures consistency, convenience, and collaboration.

ClickSafe – Phishing URL Detector

ClickSafe is a robust solution for identifying and mitigating phishing threats. This web application is designed to analyze URLs in real-time, helping users detect phishing attempts by evaluating a URL's characteristics, WHOIS data, and more. Developed using Python, Flask, and integrated with an XGBoost model, ClickSafe is ideal for users who want a quick and effective way to avoid falling victim to phishing attacks while browsing.

Samsung Prism Virtual Internship- Removing Moiré effect from Images

Developed a system for eliminating moiré artifacts from images using advanced image processing techniques, a critical problem for high-resolution imaging systems like those in Samsung PRISM. Leveraged Python and machine learning techniques to build this solution, which ensures that digital images maintain their fidelity and sharpness by eliminating unwanted patterns.
